# Elo Tracker
This project is an Elo Rating System that allows users to create a leaderboard for friend groups when playing various games.
I created this project to learn more about MySQL, OOP and how the Elo rating system works.

## Features

- Create Players
- Remove Players
- Record Matches
- Automatic Elo Calculation
- Leaderboard
- Match History
- Player Stats

  ## How to run

  1. Install Python
  2. Install the dependencies
  ```bash
  pip install -r requirements.txt
  ```
  3. Create a .env file
  ```env
  host=YOUR_HOST
  user=YOUR_USER
  password=YOUR_PASSWORD
  ```
  4. Start the project
  ```bash
  python main.py
  ```
